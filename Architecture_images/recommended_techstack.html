Language
core
Python (backend) + JavaScript (frontend)
You already know Python — use it for everything on the server. React uses JS/JSX which is easy to pick up coming from Python. No need to learn a new backend language.
Backend
core
FastAPI
The best Python web framework for building APIs. Auto-generates docs at /docs so you can test every endpoint in the browser without any extra tool. Async support built in — important for proxying LLM calls which can be slow.
You already know Python. FastAPI will feel natural immediately.
Frontend
core
React + Vite
React for the dashboard UI — components, state, API calls. Vite as the dev server: starts in under a second, no config needed. Skip Next.js — it adds SSR complexity you don't need this week.
also use
TailwindCSS + shadcn/ui
Tailwind for styling without writing CSS files. shadcn/ui gives you pre-built components (tables, forms, charts) that look professional out of the box. Your dashboard will look great without design work.
Database
core
PostgreSQL + SQLAlchemy + Alembic
PostgreSQL is the best open-source relational DB — handles users, keys, roles, usage logs perfectly. SQLAlchemy is Python's most popular ORM — write Python classes instead of raw SQL. Alembic handles DB migrations (when you change your schema, it updates the DB safely).
Your SQL knowledge transfers directly. SQLAlchemy just wraps it in Python.
Cache + Rate limiting
core
Redis
Stores rate limit counters per user — e.g. "user X has made 47 requests this minute". Much faster than hitting PostgreSQL on every single API call. Also stores sessions.
Auth
core
Google OAuth + python-jose (JWT)
Google OAuth handles the login flow — users click "Sign in with Google", Google confirms identity, your backend gets their email. python-jose creates signed JWT tokens your frontend uses on every request to prove who they are.
Encryption
core
cryptography (Python library)
Used to encrypt provider API keys before saving to DB (AES-256). One of the most trusted crypto libraries in Python. Simple API — encrypt before save, decrypt before use.
Infrastructure
deploy
Docker + Docker Compose
One Dockerfile each for backend and frontend. Docker Compose ties everything together — backend + frontend + PostgreSQL + Redis all start with one command. Same setup works locally and in the cloud.
deploy
Railway (cloud hosting)
The easiest platform to deploy a Docker project in 2025. Connect your GitHub repo, it detects Docker Compose, deploys everything. Free tier works for this project. No AWS complexity needed on week 1.
